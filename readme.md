# StudyMate
## Overview
StudyMate is a simple study-planning app that helps students organise study sessions, track completed tasks, and manage deadlines.

## Features
- Create study schedules
- Add tasks
- Mark tasks as completed
- Set reminders
- View daily progress

## Who can use it
- Students
- Teachers
- Reasearchers 

## How to use
- Create an account
- Log in
- Create a study plan
- Add subjects
- Mark tasks as completed

## Benefits
- Better organisation
- Improved productivity
- Deadline tracking 

## User policy
By using StudyMate, users agree to:
- Use the application responsibly.
- Provide correct information whe creating study plans.
